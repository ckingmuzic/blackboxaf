BlackBoxAF
Salesforce Pattern Vault — Extract the architecture. Reuse the knowledge.
Open Source MIT License AI-Powered Desktop + CLI

Every Salesforce consultant rebuilds the same patterns: approval flows, validation rules, record-triggered automation. BlackBoxAF extracts these structural blueprints from any org—automatically anonymized—and catalogs them into a searchable, gamified pattern vault. Nothing else like it exists in the Salesforce ecosystem.

Pattern Extraction

Flows, Apex, LWC, Reports, Validation Rules, Layouts, Custom Objects & Fields

Auto-Anonymization

CamelCase brand detection with Salesforce ecosystem awareness. Zero proprietary data leaks.

AI Search

Natural language pattern search powered by Claude. Ask “find approval flows with email notification.”

VS Code Extension

Browse and insert patterns directly in your editor. Search, filter, click to insert.

MCP Server

Works with Claude Code, Cursor, Windsurf, ChatGPT Desktop. AI assistants search your vault.

Agentforce-Style

Autonomous composition: describe what you need, get a multi-component solution blueprint.
